# Team Workflow

## สมาชิกในทีม
- สมาชิก 1: นายณัฐธเนศ กำเนิดกาลึม (รหัส 663380011-8) หน้าที่ ทำการเขียน bubble sort 
- สมาชิก 2: นายพศวีร์ ผองแก้ว (รหัส 663380219-4) หน้าที่ รับค่า Input
- สมาชิก 3: นางสาวพิมอัปสร แพน (รหัส 663380509-5) หน้าที่ ทำการแสดงผลลัพธ์

## กระบวนการทำงานร่วมกันของทีม
- 1.สมาชิกที่รับ Input (นายพศวีร์)
ทำหน้าที่รับค่าตัวเลขจากผู้ใช้ และตรวจสอบว่าข้อมูลถูกต้องก่อนส่งต่อให้ส่วนการเรียงลำดับ
- 2.สมาชิกที่พัฒนา Bubble Sort (นายณัฐธเนศ)
รับข้อมูลที่ได้จากส่วน Input มาทำการเรียงลำดับด้วย Bubble Sort และส่งผลลัพธ์ที่เรียงแล้วต่อไปยังส่วนแสดงผล
- 3.สมาชิกที่แสดงผลลัพธ์ (นางสาวพิมอัปสร)
รับข้อมูลที่ถูกเรียงลำดับแล้ว และแสดงผลให้ผู้ใช้เห็นอย่างถูกต้องและชัดเจน
